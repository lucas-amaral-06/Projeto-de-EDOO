Bem-vinda(o) ao SysMed C++

Um projeto de sistema de gerenciamento médico em C++, aplicando os conceitos de Programação Orientada a Objetos. Desenvolvido para a cadeira de Estrutura de Dados Orientadosa Objetos do curso de Sistemas de Informação - CIn/UFPE.

Sobre o Projeto

O SysMed C++ é um sistema de linha de comando que simula o gerenciamento de uma clínica médica. O projeto foi criado para aplicar de forma prática os pilares da Programação Orientada a Objetos.

Principais Funcionalidades

  • Agendamento de consultas.
  • Cadastro de recepcionistas, pacientes e médicos.
  • Gestão de prontuários médicos.
  • Emissão de Atestados, Evoluções e Receitas.

Tecnologias e Conceitos

  • C++: Linguagem principal do projeto.
  • POO: Encapsulamento, Herança, Polimorfismo.
  • Estrutura de Dados: Uso de vetores e classes.
  • Makefile: Automação da compilação.
